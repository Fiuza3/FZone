# 🚀 FZone ERP

> *"Gerenciamento empresarial simplificado e eficiente"*

Bem-vindo ao **FZone ERP**, o sistema que transforma a gestão da sua empresa em uma experiência simples e produtiva.

## 🤔 O que é o FZone?

É um ERP completo e intuitivo que permite:

- **Gerenciar tarefas** - Organize e acompanhe todas as atividades da equipe
- **Controlar estoque** - Mantenha o controle preciso de produtos e inventário
- **Organizar finanças** - Acompanhe receitas, despesas e fluxo de caixa
- **Administrar pessoas** - Gerencie funcionários e departamentos com facilidade

## 🛠️ Tecnologias utilizadas

- **Vue 3** - Framework frontend moderno e reativo
- **MongoDB** - Banco de dados NoSQL flexível e escalável
- **Express** - Framework backend robusto para Node.js
- **Tailwind CSS** - Framework CSS utilitário para design responsivo
- **JWT** - Autenticação segura baseada em tokens

## 🏃‍♂️ Como executar o projeto

### Pré-requisitos

- **Node.js** - Versão 14.x ou superior
- **MongoDB** - Versão 4.x ou superior
- **npm** ou **yarn**

### Instalação e execução

```bash
# Clone o repositório
git clone https://github.com/seu-usuario/fzone-erp.git
cd fzone-erp

# Configure o backend
cd backend
npm install
cp .env.example .env
# Edite o arquivo .env com suas configurações
npm run dev

# Configure o frontend
cd ../frontend
npm install
npm run dev
```
